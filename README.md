# Prime-pick 🛒

Prime-pick is a modern e-commerce web application built with **React + Vite + Tailwind CSS**.

## ✨ Features

- Product browsing and filtering
- Product details with image gallery
- Ratings and reviews
- Cart and checkout flow
- Dark mode UI
- Responsive design

## 🛠️ Tech Stack

- React JS
- Vite
- Tailwind CSS
- React Router
- Context API

## 🚀 How to Run Locally

```bash
npm install
npm run dev
```
